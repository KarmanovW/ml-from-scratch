
Linear Regression — базовая модель регрессии, аппроксимирующая зависимость целевой переменной линейной комбинацией признаков.
Реализовано обучение методом градиентного спуска, а также оценка качества по метрикам вроде MSE / RMSE.

Logistic Regression — модель бинарной классификации, использующая sigmoid-функцию для получения вероятности принадлежности к классу.
Реализовано обучение через оптимизацию log-loss (кросс-энтропии) с помощью градиентного спуска, а также получение предсказаний и вероятностей.
<img width="1385" height="918" alt="image" src="https://github.com/user-attachments/assets/ac7af0ec-63b9-4646-8f4c-510b88a80526" />

Decision Tree — алгоритм машинного обучения для классификации и регрессии, который строит дерево решений, последовательно разбивая данные по признакам.
В реализации использовались критерии качества разбиения Gini impurity / entropy, рекурсивное построение дерева и ограничения глубины для борьбы с переобучением.
<img width="2140" height="1006" alt="image" src="https://github.com/user-attachments/assets/7a3e36c6-8495-4e91-aebe-be64d58c13fc" />

Random Forest — ансамблевый метод, основанный на обучении множества деревьев решений на случайных подвыборках данных и признаков.
Такой подход уменьшает переобучение и повышает устойчивость модели по сравнению с одиночным деревом.
